# Firmware Distribution Governance Adoption

This repository adopts AI-Native Engineering Operating System **2.2** from
`pcvantol/djconnect/docs/governance/PLATFORM_ARCHITECT_SYSTEM_INSTRUCTIONS.md`.
There are no local exceptions.

Definition of Done requires synchronized clean main, implementation-reality
check, manifest JSON/asset-name/checksum/release-note validation, updated
rolling records/immutable history and one reviewable PR. This repository has no
source-runtime coverage obligation.

Release profile: public firmware manifest, binaries, SHA256 checksums and
GitHub Release evidence. Docker, Cloudflare Workers and HACS are not targets.
